What is our project about?

Our project is a simple but quite interesting quiz for new and old Dota2 players.

Why Dota2?

In addition to the interest of the majority of our team in this game, we also had the goal of bringing something interesting, but at the same time familiar in our mobile application.

Why quiz format?

We decided that the quiz format would be suitable for implementing our ideas and using all the elements required for the project.

What did we use for the project?

Recycler View

it is a UI component that allows us to create a scrollable list

Data Binding

allows you to bind UI components in layouts to data sources in your application using a declarative format

Networking

lets your application
connect to the internet or any other local network and allows you to perform network operation

Other elements

For the project, we used many other elements and functions, including radiobuttons, buttons, event handlers and etc.
